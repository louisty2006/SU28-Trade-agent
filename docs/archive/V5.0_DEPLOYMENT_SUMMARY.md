## 🎯 系统能力

### 五层防护系统
```
数据验证 → LLM防幻觉 → 输出验证 → 最终审视 → 人工确认
```
**理论错误率**：每层过滤90%错误 → 最终 ≈ 0.001%（万分之一）

### 五大AI方向
1. ✅ 图表型态识别
2. ✅ 知识图谱 + 因果推理
3. ✅ LLM情绪分析
4. ✅ Multi-Agent协作
5. ✅ 霊視记忆

### 三大原则优化
- **赚最多**：图表型态 + Multi-Agent + 记忆学习
- **赚最快**：即时监控 + 因果推理
- **风险最少**：五层防护 + 风险分析师 + 强制停损

## 🚀 立即使用

### 1. 安装依赖
```bash
cd /Users/lautinyam/stock_scanner
pip install -r requirements.txt
```

### 2. 配置（可选）
```bash
# 复制配置模板
cp config.yaml.example config.yaml

# 编辑配置（添加LLM API Key等）
nano config.yaml
```

### 3. 运行
```bash
# 查看统计
python main_v5.py --stats

# 运行每日分析
python main_v5.py --daily

# 启动即时监控
python main_v5.py --monitor
```
