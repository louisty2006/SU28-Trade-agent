## 🏗️ 系统架构

### 五层防护系统

```
第一层：数据验证（多源交叉验证、合理性检查）
    ↓
第二层：LLM防幻觉（强制引用来源、自我质疑）
    ↓
第三层：输出验证（逻辑检查、数字检查）
    ↓
第四层：最终审视（审计员角色、只检查不判断）
    ↓
第五层：人工确认（异常清单、一键验证）
```

### 五大AI方向分析

1. **图表型态识别**：规则侦测 + LLM看图验证
2. **知识图谱 + 因果推理**：累积式关系资料库
3. **LLM情绪分析**：新闻与市场情绪
4. **Multi-Agent协作**：4个分析师角色
5. **霊視记忆**：经验累积与学习

## 🚀 快速开始

### 安装依赖

```bash
pip install -r requirements.txt
```

### 运行每日分析

```bash
python main_v5.py --daily
```

### 启动即时监控

```bash
python main_v5.py --monitor
```

### 查看统计

```bash
python main_v5.py --stats
```

### 运行回测（Orchestrator 对接）✨ 新增

```bash
python main_v5.py --backtest 2025-01-01 2025-01-31
```
