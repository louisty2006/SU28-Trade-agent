## 🚀 如何使用

### 步驟 1：查看數據源說明
```bash
cat DATA_SOURCES.md
```

### 步驟 2：註冊 API Keys
按照 `API_KEYS_GUIDE.md` 註冊（推薦註冊 5 個）

### 步驟 3：填入 .env
```bash
cp .env.example .env
nano .env  # 填入 API Keys
```

### 步驟 4：計算配額
```bash
python quota_calculator.py
```

### 步驟 5：開始使用
```bash
python main.py --all
```
